# Project Overview

Welcome to the **Bark Blinks & Solana Actions** project! This project integrates Solana blockchain actions with a modern web interface, enabling seamless payments, QR code management, and donations.

## Key Features

- **Solana Payments**: Effortlessly send and receive SOL and SPL tokens.
- **QR Codes**: Generate and scan QR codes for quick and secure transactions.
- **Donation Management**: Contribute to causes directly on the Solana blockchain with ease.
